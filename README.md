# CyberForenxTasks

This repository contains three frontend web development projects built using HTML, CSS, and JavaScript.

## Task 1: Portfolio Website

A responsive personal portfolio website designed to showcase skills, projects, education, and contact information. The project focuses on creating a professional online presence with a clean and modern user interface.

### Features

* Responsive design for all devices
* About Me section
* Skills showcase
* Projects section
* Contact information
* Modern and user-friendly UI

### Technologies Used

* HTML5
* CSS3
* JavaScript

---

## Task 2: Shopping Cart

A simple e-commerce shopping cart application that allows users to browse products, add items to the cart, update quantities, and view the total price dynamically.

### Features

* Product listing
* Add to cart functionality
* Quantity management
* Dynamic total price calculation
* Responsive design

### Technologies Used

* HTML5
* CSS3
* JavaScript

---

## Task 3: Toaster Notification Application

A web application that displays dynamic toast notifications for different user actions. The project demonstrates DOM manipulation and interactive UI components using JavaScript.

### Features

* Success notifications
* Error notifications
* Warning notifications
* Information notifications
* Auto-dismiss functionality
* Smooth animations
* Responsive design

### Technologies Used

* HTML5
* CSS3
* JavaScript
